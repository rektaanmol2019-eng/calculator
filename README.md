# Advanced Calculator

## Overview
A professional **scientific calculator** with basic arithmetic (+, −, ×, ÷), percentage (%), square root (√), memory functions, calculation history, error handling, and smooth animations.[1][2]

## How to Start
1. Place these files in one folder:  
   - `index.html`  
   - `style.css`  
   - `script.js`[3][2][1]
2. **Double‑click `index.html` to open in Chrome, Edge, or Firefox**.  
   No server or build required.[3]

## Features
- **Operations**: +, −, ×, ÷, %, √ (with smart percentage based on previous operation).[2]
- **Display**: Previous/current operand, animated error messages (auto‑clear), shimmer effect.[1][2]
- **History**: Stores last 50 calculations with timestamp; toggleable view.[2]
- **Memory**: M+, M−, MR, MC with visual indicator and haptic feedback.[1][2]
- **Error handling**: Divide by zero, invalid √, duplicate decimals with friendly messages.[2]
- **UI**: Glassmorphism design, gradients, floating animations, button press effects.[1]
- **Precision**: 12‑decimal rounding for accuracy.[2]

## Quick Usage
1. **Numbers/Decimal**: Click digits or `.` to enter values.[2]
2. **Operations**: Click +, −, ×, ÷, %, √.[2]
3. **Calculate**: Click `=` to compute.[2]
4. **Memory**: M+ (add to memory), M− (subtract), MR (recall), MC (clear).[2]
5. **Controls**: AC (clear all), DEL (backspace), history toggle.[3][2]
6. **Errors**: Shown temporarily (e.g., "Cannot divide by zero") with vibration.[2]
